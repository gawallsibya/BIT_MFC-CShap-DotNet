문제 10개가 출제되고 사용자에 의해서 문제를 풀게 된다.
10개의 문제를 다 풀게 되면 결과를 저장하고 저장된 결과를 볼 수 있다.
기능 1.  문제 출제 : 랜덤하게 5개의 문제를 출제하게 된다.(아래 소스 참조)
	#include <stdio.h>
	#include <conio.h>	// getch
	#include <stdlib.h>	// srand
	#include <time.h>	// time
	void main()	{
		while(1)  {
			system("cls");  	        srand(time(NULL));
			printf("%d\n", rand()%100);   system("pause");
		}
	}
  기능 2. 문제 풀이 
	출제된 문제가 하나씩 화면에 출력되고 사용자 입력을 통해 문제를 풀게 된다.
	문제를 푸는 도중에 프로그램 종료는 불가하다.            
  기능 3. 결과 확인 : 문제 풀이 결과를 확인할 수 있다.
             ▶ 출제 문제 :  5문항
             ▶ 정     답 :  3문항
             ▶ 오     답 :  2문항
             ▶ 세부 내용   
             -------------------------------------
             번호    문제         입력값	 정답여부
             --------------------------------------
              [1]    1 + 3       100            X            
	--------------------------------------
