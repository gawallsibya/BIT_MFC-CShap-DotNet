  C# 코드에서 다양한 모듈을 사용하는 Sample 코드 구현....

실습1. WinForm 프로젝트 생성 및 tab 컨트롤 생성

실습2. 첫번째 tab 컨트롤( 기존 dll 사용하기 )
        계산기 기능을 가지고 있는 계산기 dll을 만들고 
        해당 기능을 사용한 계산기 UI를 구성할 것

실습3. 두번째 tab 컨트롤( 어셈블리 사용하기 )
         계산기 기능을 가지고 있는 어셈블리를 만들고
         실습2와 동일한 UI를 구성하여 사용한다.

실습4. 세번째 tab 컨트롤( com component 사용하기 )
         계산기 기능을 가지고 있는 com component를 만들고
         실습3과 동일한 UI를 구성하여 사용한다.

실습5. 네번째 tab 컨트롤( ActiveX 컨트롤 사용하기 )
         COM 수업에 만들었던 ActiveX컨트롤을 사용하여
         UI를 구현하고 실행시킨다.
